Horisontell projektion (fx):–
Längd = höjd / sin(v)
fx = längd · cos(v) = höjd / tan(v).

Hur du använder måtten ute på jobbet

  1. Välj 0-punkt. Markera på röret var det ska hamna i slutläget.
  2. Skriv in höjd + vinkel. Appen räknar L och fx.
  3. Mät/markera. Följ dina markeringar och bocka till vinkeln.
Tryck en riktning för att lägga till ett rörsteg.
Diagonal steg (valfri vinkel) D az el mm
Azimut: 0=E, 90=N. Elevation: +upp / -ner.
Ex: 45° i plan → D 45 0 126 • 45° upp i E → D 0 45 126
Format per rad:
• E/W/N/S/UP/DOWN + mm (ex E 120)
• D az el mm (ex D 45 0 126)
